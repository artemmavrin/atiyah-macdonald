\begin{exercise}
Let \(A\) be a Noetherian ring and let \(f = \sum_{n=0}^\infty a_n x^n \in A[[x]]\).
Prove that \(f\) is nilpotent if and only if each \(a_n\) is nilpotent.
\end{exercise}

\begin{solution}
(\(\Rightarrow\)) This direction follows from \expartref{1}{5}{ii}, which does not use the Noetherian assumption.


\end{solution}
